# AI Stock Advisor - Frontend

This is the React frontend for the AI Stock Advisor application.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:3000`.

## Features

- **Dynamic Navigation**: Sidebar for history and profile access.
- **Modern UI**: Dark-themed dashboard with sleek glassmorphism effects.
- **Real-time Charts**: Stock price history visualization.
- **AI-Powered Insights**: Detailed analysis reports rendered from Gemini responses.

For more details, see the [root README](../README.md).
