# AI Stock Advisor

An AI-powered stock analysis platform that provides deep insights using freely available financial data and Google's Gemini models. The application combines real-time financial data from `yfinance` with the advanced reasoning capabilities of `gemini-2.5-flash-lite` to provide comprehensive stock analysis.

## Features

- **Real-time Fundamentals**: Fetch the latest market cap, P/E ratios, growth metrics, and analyst recommendations.
- **Historical Trends**: Visualization of stock price history over the last 5 years.
- **AI Analysis**: Deep-dive analysis powered by Gemini, including growth assessment, value analysis, and news sentiment.
- **Google Search Integration**: Grounded AI responses using live web data for the most up-to-date sentiment.

## Architecture

- **Frontend**: React + TypeScript + Vite + Tailwind CSS
- **Backend**: FastAPI (Python)
- **AI Model**: Google Gemini 2.5 Flash Lite
- **Data Source**: Yahoo Finance (yfinance)

## Setup & Installation

### Prerequisites

- [Python 3.13+](https://www.python.org/)
- [Node.js 18+](https://nodejs.org/)
- [uv](https://github.com/astral-sh/uv) (Recommended for Python package management)
- A Google Gemini API Key from [Google AI Studio](https://aistudio.google.com/)

### 1. Backend Setup

```bash
cd backend
```

Create a `.env` file in the `backend` directory:
```env
GEMINI_API_KEY=your_api_key_here
```

Install dependencies and run:
```bash
uv run main.py
```
The backend will start at `http://localhost:8000`.

### 2. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
The frontend will start at `http://localhost:3000`.

## Usage

1. Open `http://localhost:3000` in your browser.
2. Enter a stock ticker (e.g., `AAPL`, `NVDA`, `TSLA`).
3. Click "Analyze Stock".
4. View the comprehensive report generated by the AI advisor.

## Technical Details

- **Backend Logic**: Separated into routes, services, and prompts for modularity.
- **Model Configuration**: Uses `gemini-2.5-flash-lite` by default with Google Search grounding enabled.
- **CORS Handling**: Configured to allow requests from the local development server.
